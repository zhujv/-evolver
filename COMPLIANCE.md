# Compliance Notes

This repository is intended to be independently developed and maintained.

## Open-Source Acknowledgements

This project acknowledges open-source inspirations from:

- OpenClaude
- OpenCode
- HermesAgent

These acknowledgements refer to open-source ideas and public implementations as learning references. This repository maintains its own implementation and should respect each upstream project's license and attribution terms.

## Source Code Provenance

- The project does not include proprietary source code from third-party closed products.
- Integrations with model providers are implemented through public APIs and SDKs (for example, OpenAI and Anthropic official interfaces).
- Architectural patterns such as agent loops, tool registries, and RPC transport are common software design patterns and are implemented in this repository's own code.

## Contributor Requirements

- Do not copy code from proprietary products, private repositories, or paid-only artifacts.
- If referencing public open-source projects, preserve required license and attribution obligations.
- Keep prompts, docs, and comments neutral about "source inheritance" unless a verified, licensed dependency is explicitly documented.

## Licensing Boundary

- Third-party libraries remain under their respective licenses.
